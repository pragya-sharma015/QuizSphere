🚀 QuizSphere – Smart Quiz & Evaluation Platform

QuizSphere is a full-stack web application built with ASP.NET Core that enables teachers to create, manage, and analyze quizzes while allowing students to attempt them in a structured, time-bound, and competitive environment.

It is designed with a dual-portal system (Teacher & Student) and supports department-based quiz organization, real-time scoring, and dynamic leaderboards.

📌 Features
🔐 Authentication & Roles
Secure login and registration
Role-based access (Teacher / Student)
🧑‍🏫 Teacher Portal
Create and manage quizzes
Add multiple-choice questions with correct answers
Set quiz start time, end time, and time limit
View quiz analytics (attempts, scores, performance)
👩‍🎓 Student Portal
Browse quizzes by department
Attempt quizzes with countdown timer
Auto-submit on time expiry
View scores instantly
🏆 Leaderboard System
Global ranking of students
Department-wise leaderboard
Filter by quiz and teacher
Identify top performers
